## Color Palette
- Primary Green: #138808 (Government Green)
- Saffron: #FF9933
- White: #FFFFFF
- Light Blue: #87CEEB
- Dark Text: #333333
- Light Gray: #F5F5F5

## Key Features to Implement
1. **Mobile-First Design**
2. **Fast Loading (< 3 seconds)**
3. **WhatsApp Integration**
4. **Google Maps Integration**
5. **Image Gallery Slider**
6. **Contact Form**

## Content Guidelines
- Use simple, clear language
- Include Hindi translations for key terms
- Focus on trust and reliability
- Highlight government affiliation
- Show local community focus
